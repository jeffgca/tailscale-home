# Tailscale Home

A browser extension for Tailscale / Tailnet users that authenticates you gives you access to your tailnet status, devices and services.

1. a preferences page the user can use to enter their tailscale api key, including storage of the api key across browser sessions.

2. a browser action button that opens the main extension page

3. the main extension page that displays a list of tailnet devices from the tailscale api, including the name, ip address and whether the device is currently online

4. a background timer that regularly checks to see if the tailscale network is available
